InfinityFree — загрузка Divane Client
=====================================

Домен: divaneclient.gt.tc
Аккаунт: if0_42343075

СПОСОБ 1 — File Manager (проще)
--------------------------------
1. В панели InfinityFree нажми "File Manager" (оранжевая кнопка).
2. Открой папку htdocs (это корень сайта).
3. Удали стандартные файлы (index2.html и т.п.), если они мешают.
4. Загрузи ВСЁ из папки upload-to-infinityfree:
   - index.html
   - app.js
   - style.css
   - launcher-auth.html
   - папку assets (целиком)

5. Открой в браузере: https://divaneclient.gt.tc
   (DNS может обновляться до 72 часов)

СПОСОБ 2 — FTP (FileZilla)
--------------------------
Хост:     ftpupload.net
Логин:    if0_42343075
Пароль:   пароль от аккаунта InfinityFree
Порт:     21
Папка:    /htdocs/

Перетащи туда те же файлы, что и в способе 1.

ВАЖНО — API авторизации
-----------------------
InfinityFree НЕ запускает Node.js.
Регистрация/вход на сайте заработают только после PHP API.

Следующий шаг: загрузить api/ в htdocs/api/ (будет добавлено отдельно).

Пока API не настроен:
- Сайт откроется (главная, дизайн)
- Кнопки "Войти" / "Регистрация" не будут работать

Лаунчер локально может использовать localhost:3000 как API,
или позже — https://divaneclient.gt.tc/api
